What is Occupator?

Occupator is a website which locates jobs and homeless shelters mainly for people who don't have homes. It is available in many different languages.

JOB FINDER

The job finder searches for nearby jobs in your skillset and even lets you contact the employer. It uses the Indeed.com API

SHELTER SEARCHER

The Shelter Searcher searches for neaby homeless shelters with free beds etc. It uses the Google Maps API

How does it work?

Occupator uses APIs to locate jobs and holeless shelters (One API from Indeed.com, one Google Maps API)
